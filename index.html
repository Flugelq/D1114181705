什麼是 HTML？

HTML Tutorial

HTML（超文本標記語言）是用於創建網頁和網頁應用程序的標記語言。

HTML 的基本結構

一個基本的 HTML 文件包含以下幾個部分：

  • DOCTYPE聲明: 定義文檔類型。
  • html 標籤: 整個 HTML 文件的根元素。
  • head 標籤: 包含元數據，如標題和樣式。
  • body 標籤: 包含實際顯示在網頁上的內容。

基本標籤示例

以下是一些常用的 HTML 標籤：

  • <h1></h1>: 主要標題。
  • <p></p>: 段落。
  • <a href="url"></a>: 鏈接。
  • <img src="image.jpg" alt="描述">: 圖片。

為什麼要學習 HTML？

學習 HTML 是進入網頁開發的第一步，它幫助你理解網頁是如何結構化的，並為後續的 CSS 和 JavaScript 打下基礎。

結論

無論你是想建立個人網站還是成為專業的網頁開發者，了解 HTML 都是必不可少的。
